My personal blog.

`https://mason-lee-101.github.io/`

## Blog posts

This site is built with Jekyll on GitHub Pages. Blog pages live as markdown files under `blogs_posts/`.

Each post needs front matter like this:

```md
---
layout: post
title: Post title
date: 2026-05-21
description: Short preview text
---
```
